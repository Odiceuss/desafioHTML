Primeiras máquinas

(voltar)

Colossus (1943)

O colossus foi um computador ou melhor, um conjunto de computadores desenvolvidos por estudiosos britânicos nos anos de 1943 a 1945 para ajudar na criptoanálise da cifra de Lorenz (Uma máquina nazista). Colossus utilizava válvulas termiônicas (a vácuo) realizando operações booleanas e de contagem.

ENIAC (1946)

O ENIAC foi o primeiro computador eletroeletrônico digital de grande escala, criado pelos cientistas norte-americanos John Eckert e John Mauchly, feito para decifrar trajetórias táticas que exigiam conhecimento substancial em matemática com mais agilidade que um ser humano.

Havard Mark I

Criada em (1994) o Havard Mark I (ASCC) foi a primeira calculadora eletromecânica automática produzida em larga escala desenvolvido nos EUA, projetada pelo estudante de física Howard Aiken com parceria com a empresa IBM.

Como surgiu a internet

(voltar)

A internet surgiu, na década de 1960, como uma rede (chamada Arpanet) para o envio de informações entre o centro de pesquisas e intalações militares e o Pentágono. A primeira mensagem enviada foi feita entre computadores da UCLA e da SRI interessante saber também que foi a primeira perda de pacotes na internet, onde a mensagem enviada foi a palavra "Login" onde chegou ao destino somente a palavra "Lo".

Browsers(Navegadores)

(voltar)

Browsers, ou navegadores de internet, são programas usados para acessar e visualizar páginas da web. Eles interpretam o código HTML, CSS, JavaScript e outros tipos de arquivos e exibem o conteúdo de sites de maneira legível e interativa. Exemplos populares de navegadores incluem:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari
  • Opera

Eles também oferecem recursos como navegação por abas, gerenciamento de downloads, histórico de navegação, e suporte a extensões, que são funcionalidades extras que você pode adicionar ao navegador.

Sobre HTML

(voltar)

HTML (HyperText Markup Language)

é a linguagem de marcação usada para estruturar e organizar o conteúdo de páginas da web. Ele define a estrutura básica de um site usando tags para elementos como textos, imagens, links, listas, botões, e muito mais. O HTML é o alicerce de toda página web e é interpretado pelos navegadores para exibir o conteúdo de maneira adequada.

Tags

(voltar)

Tags são pequenos comandos para estrututar seu HTML, elas são utilizadas para dar forma e posição a cada tipo de elemento na construção do seu projeto HTML, existem vários tipos de tags, para textos, imagens, links entre outros, as mais utilizadas em um projeto são:

  1. < h1 > (define um título)
  2. < p > (define um parágrafo)
  3. < a > (define um link)
